My reflection

While working on this project, I learned how powerful HTML, CSS, and CSS frameworks such as Bootstrap can be. The complexity of these tools can be a bit overwhelming if one does not stay organized while coding, due to the sheer amount of text on the screen. It's very easy to lose track of which element is closed by which /div tag. Previous to this, I did not even know what Bootstrap was--I'd heard of it--but now I know how approachable and useful it is. I also learned the value of coding a single page well, and then reusing that page and just changing the pieces that need to be updated. This has been a very enjoyable learning experience--I just wish I had more time to spend investigating Bootstrap and other CSS frameworks!
